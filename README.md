# Skyscanner 智慧篩選助手
自動過濾 Skyscanner 超標價格與時間的 Chrome 插件

這是一個 Chrome 擴充功能，專門用於優化 Skyscanner 的搜尋體驗。

### 主要功能
- **價格過濾**：自動隱藏超過預算的機票卡片。
- **時間篩選**：自動隱藏出發或回程晚於指定時間的航班。
- **自動標示**：符合條件的機票會以青色邊框顯現。

### 安裝方法
1. 下載此專案程式碼。
2. 開啟 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右側的「開發者模式」。
4. 點擊「載入解壓縮過濾擴充功能」，選擇此資料夾。
